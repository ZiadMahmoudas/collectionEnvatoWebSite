# Hello There, My Name's Ziad

**How To Work With This App:**

1. `node -v`  
   To check the installed Node.js version.

2. **Download Node.js**  
   If Node.js is not installed, download it from [Node.js official website](https://nodejs.org/).

3. `npm -v`  
   To check the installed npm (Node Package Manager) version.

4. `npm install -g npm`  
   To install or update npm globally.

5. `npm install -g @angular/cli`  
   To install Angular CLI globally and use Angular commands.

6. **When you pay for this template, before running it, you must run:**  
   `npm install`  
   This will install all necessary dependencies for the project.

7. `ng s -o`  
   To run the Angular application and open it in your default browser.
